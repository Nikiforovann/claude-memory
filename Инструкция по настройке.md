# Инструкция по настройке персонального ассистента

## 1. Постоянная папка памяти — готово

Папка `Для клода` подключена к Cowork, и в ней лежат:
- `CLAUDE.md` — краткая справка обо мне (домены, предпочтения)
- `memory/` — глоссарий, контекст по доменам, папки для людей и проектов

Этот репозиторий (`claude-memory`) — бэкап/версия тех же файлов на GitHub.

## 2. Авторизация коннекторов

Статус: подключены и работают — Gmail, FMP, GitHub.
Не авторизованы — Slack, Notion, Asana, Atlassian, ClickUp, Linear, Monday,
BigQuery, Bigdata.com, Daloopa.

Шаги: claude.ai/customize/connectors → "+" → найти сервис → "Connect" → войти
и подтвердить доступ → включить тумблером.

## 3. Instructions for Claude (аккаунтные настройки)

claude.ai → инициалы внизу слева → Settings → "Instructions for Claude".
Применяется во всех чатах на claude.ai, не только в Cowork.

## 4. Наполнение памяти

Через диалог: "запомни, что Х значит Y" — я сам обновлю файлы.
Вручную: правь CLAUDE.md и memory/glossary.md напрямую.

Gmail проверен на предмет авто-извлечения контактов/проектов — ящик оказался
личным, полезного контента не найдено.
